## Setup

1. Install [Miniconda](https://docs.conda.io/en/latest/miniconda.html)

2. Run the setup script for your platform:
   - **Linux/macOS:** `bash setup.sh`
   - **Windows:** Double-click `setup.bat`

3. Activate the environment:
   - **Linux/macOS:** `conda activate astar`
   - **Windows:** `conda activate astar`

4. Run: `python run.py`

## Known Issues

**Windows high-DPI displays:** If the UI appears incorrectly sized, 
this is handled automatically. If issues persist, set display scaling 
to 100% in Windows display settings.
